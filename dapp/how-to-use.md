# How to Use the DApp

The BaseTrove DApp is your direct interface with the Vault. It lets you sell $TROVE at the guaranteed floor price, monitor the protocol's real-time stats, and verify every claim the team makes.

---

## Accessing the DApp

> **URL:** [basetrove.io/dapp](https://basetrove.io/dapp) *(published at launch)*

The DApp runs directly in your browser. No downloads, no installs.

---

## Requirements

- A Web3 wallet with Base network support (MetaMask, Coinbase Wallet, Rabby, etc.)
- $TROVE tokens in your wallet
- A small amount of ETH on Base for gas (transactions cost fractions of a cent)

---

## Adding Base Network to Your Wallet

If you haven't used Base before, add it to MetaMask:

| Setting | Value |
|---|---|
| Network Name | Base |
| RPC URL | https://mainnet.base.org |
| Chain ID | 8453 |
| Symbol | ETH |
| Block Explorer | https://basescan.org |

Or simply visit [chainlist.org](https://chainlist.org) and search "Base" to add it in one click.

---

## Selling $TROVE to the Vault

1. **Connect your wallet** — click "Connect Wallet" and approve the connection in your wallet app
2. **Check the current floor price** — displayed prominently on the DApp. This is the ETH you'll receive per TROVE
3. **Enter the amount** — type how many $TROVE you want to sell, or click "MAX"
4. **Review the output** — the DApp shows exactly how much ETH you'll receive before you confirm
5. **Approve the transaction** — first-time users need to approve the TROVE contract to interact with the Vault (one-time gas cost)
6. **Confirm the sell** — sign the transaction in your wallet. Your TROVE is burned and ETH arrives in your wallet in the same transaction

There are **no fees on Vault sales.** What you see is what you get.

## DApp Dashboard

The Stats section shows live on-chain data:

| Metric | Description |
|---|---|
| **Vault ETH** | Total ETH currently held in the Vault |
| **Floor Price** | Current guaranteed redemption price (ETH per TROVE) |
| **Circulating Supply** | TROVE tokens currently in circulation |
| **Total Burned** | All-time permanent burns |
| **Last Buyback** | Pool, amount, and timestamp of last mining buyback |
| **Mining Wallet Balance** | Live BTC + ETH balance of the buyback wallet |
| **Daily Vault Growth** | ETH added to Vault in last 24 hours |

All figures are pulled directly from on-chain sources — no backend, no manipulation possible.
